# ORIGAMI GRP 公式サイト

ORIGAMI GRP の公式Webサイト用リポジトリです。

---

## ■ 概要

本リポジトリは、ORIGAMI GRPの公式サイト（静的HTMLサイト）のソースコードを管理しています。

主な内容：
- AIスクール紹介ページ
- 会社概要ページ
- コンタクトページ
- FAQページ

---

## ■ 技術構成

- HTML（静的サイト）
- CSS（styles.css）
- 画像ファイル（assetsフォルダ）

フレームワークは使用していません。

---

## ■ デプロイ

本サイトは Vercel を利用してデプロイしています。

GitHubの `main` ブランチへ push すると、
Vercel側で自動デプロイされます。
